# Data Availability

This project uses public processed supplements from GEO accession `GSE131907`.

GEO notes that raw sequencing data are not openly available for this record because of patient privacy restrictions. Processed expression matrices and cell annotations are available on the GEO series page.

Source:

- https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE131907
